
Prerequistes ::

1. Install node.js [https://nodejs.org/en/download/]
2. Install express.js [$ npm install express --save]
3. Install Mongodb version 2.4.6. Create a database with name cinema . Creat a collection with name films.


Run the server:
In the current directory launch the following command ::: node server.js

client Frond-end:
The web front end is available at :: http://localhost:3000/api/front/
